# CKNCT-1 网络寻线仪使用说明

CKNCT-1 网络寻线仪主要用于普通寻线、交换机寻线、网线对线检测等场景。设备采用数字式寻线信号，具有抗干扰、无噪音、定位快捷等特点，适用于通信线路安装维护、综合布线和网络故障排查。

![CKNCT-1 网络寻线仪外观](/assets/image/cknct1/cknct1-overview.png)

## 使用前注意事项

::: warning 安全提醒
本产品为弱电测试设备，所有外围接口请勿连接强电线缆。主机接口最大可接 DC 60V 以下 POE 交换机，禁止接入交流市电、220V 电源线或其他不确定强电线路，否则可能损坏设备，甚至引起安全事故。
:::

- 请使用正规厂家提供的充电器和线材，充电要求为 **DC 5V / 2A**。
- 请勿让产品受到雨淋或受潮。
- 产品工作异常时，请联系销售商或服务中心，不要私自拆卸或改装。
- 避免在振动、冲击、强电磁干扰、极热、极冷或高湿环境中使用。
- 设备应存放在干燥、无腐蚀性气体环境中，避免阳光直射。
- 操作人员应具备低压布线和低压电子线路接线基础知识，并能读懂本说明内容。

## 产品功能特点

- 支持普通寻线、交换机寻线两种寻线模式。
- 支持网线 1–8 芯通断和线序测试。
- 支持在发射主机通过网线连接交换机或路由器状态下检测网线 1–8 芯通断情况。
- 支持 POE 交换机带电线序、带电线对显示。
- 支持通过接收器网口灯状态判断标准 POE 与非标准 POE 交换机：标准 POE 通常为网口灯常亮，非标准 POE 通常为网口灯闪烁。
- 主机与接收器采用卡槽拼接组合式结构，便于携带和收纳。
- 内置可充电锂电池，相比传统 9V 叠层电池可降低使用成本。

## 电量检测

初次收到仪器时，请确认包装完好、部件齐全。主机上下拨动功能选择开关，电量指示灯会亮约 1.5 秒；接收器打开 LED 照明灯开关，电量指示灯也会亮约 1.5 秒。根据亮灯情况可判断当前电量。

![CKNCT-1 电量指示灯](/assets/image/cknct1/cknct1-battery.png)

::: tip 提示
电池电量只在开机后显示约 1.5 秒。若需要再次查看电量，请重新拨动功能选择开关并重新启动。若“25% 电量”指示灯规律闪烁，表示电量较低，需要充电后再使用。
:::

## 网线对线功能

网线对线功能用于判断网线 1–8 芯的通断和线序。

![CKNCT-1 网线对线连接示意](/assets/image/cknct1/cknct1-cable-test.png)

操作步骤：

1. 将主机功能选择开关拨到 **“对线”** 位置，此时红色状态指示灯闪烁。
2. 将网线一端连接到主机上方 RJ45 接口，另一端连接到接收器底部 RJ45 接口。
3. 对线时接收器不用开机。
4. 观察主机与接收器两端指示灯亮灯情况，判断网线 1–8 芯的通断和线序。

## 寻线功能

CKNCT-1 采用数字式寻线，具备抗干扰、无噪音特点，支持普通网线寻线和交换机寻线两种模式，接收器灵敏度可调。

![CKNCT-1 寻线功能连接示意](/assets/image/cknct1/cknct1-tracing.png)

操作步骤：

1. 将主机功能选择开关拨到 **“寻线”** 位置。
2. 状态指示灯绿色闪烁时为普通寻线模式。
3. 按 **“模式”** 按键可切换寻线模式；状态指示灯红绿交替闪烁时为交换机寻线模式。
4. 将需要寻找的网线或电话线接到主机上方对应接口；其他线材可通过 RJ45 转 BNC 头或鳄鱼夹进行连接。
5. 按住接收器中间的寻线按钮，将灵敏度调节拨片调至最大。
6. 将接收器尖端寻线探测头靠近待查线缆堆。
7. 如果寻线器发出周期性声音，说明当前线缆为目标线缆。
8. 如果附近线缆较多并同时响起，可适当调低灵敏度，以更准确定位被查线缆。

### 普通寻线模式

普通寻线模式适用于寻找另一端没有任何连接的网线、同轴视频线、电源线或其他类型的两芯线缆。

### 交换机寻线模式

交换机寻线模式适用于目标线缆连接交换机或路由器的场景。使用时可将接收器灵敏度拨到带载位置附近，寻线效果更好。

对于某些 POE 交换机，由于网口内部电路存在连接，目标线附近的网线也可能接收到寻线信号并发声。此时请按主机 **“模式”** 键切换到交换机寻线模式，并适当下调灵敏度，直到准确定位目标线缆。

::: warning POE 与强电注意事项
主机可耐压 DC 60V，可带电直连目前市面上部分 POE 交换机，但这并不代表仪器可以接入强电设备。接入不确定电源线前，必须确认线路没有交流电或市电 220V，否则会烧坏设备，甚至可能引发电气危险。
:::

## 维护与存放

- 请保持接口清洁，避免灰尘、油污和水汽进入接口。
- 长时间不用时，请将设备存放在干燥、无腐蚀性气体环境中。
- 避免阳光直射和剧烈撞击。
- 请勿私自拆卸或改装设备。
- 低电量时请及时充电，建议使用符合要求的 DC 5V / 2A 充电器。
