# CKIPC 高清监控测试仪使用说明

CKIPC 高清监控测试仪是一款面向安防监控安装、网络摄像机调试和综合布线维护的多功能测试仪。

设备采用智能操作系统和 7 英寸高清触摸屏，支持网络摄像机、模拟摄像机、同轴高清摄像机、PoE 供电、网线测试、TDR 断点测试、光功率测试和红光故障定位等功能。

> 本页面根据 CKIPC 快速使用指南整理。不同硬件版本、软件版本和选配功能可能存在差异，请以设备实际界面为准。

::: warning 安全提示
- 所有接口均不得连接强电线路；
- TDR 测试接口只能连接规定的测试盒、寻线器或被测线缆；
- UTP 寻线接口不得连接交流电或超过允许电压的线路；
- 使用 PoE、电源输出功能前，必须确认被测设备的供电规格；
- 红光源为可见激光，不得直视激光输出口或光纤端面；
- 请使用原装或符合规格的充电器。
:::

## 产品外观

![CKIPC 高清监控测试仪主机](/assets/image/ckipc/ckipc-device.png)

主要用途：

- 网络摄像机 IP 探测和实时预览；
- ONVIF 摄像机搜索、登录和参数设置；
- CVBS、AHD、TVI、CVI 模拟及同轴高清视频测试；
- HDMI 视频输入；
- PoE 电压、供电线序和供电输出测试；
- 网络扫描、PING、DHCP 和网口闪烁；
- RS485 串口数据收发；
- 网线通断、线序、线长和断点测试；
- 光功率计和红光故障定位；
- 拍照、录像、回放和测试报告生成。

## 接口说明

### 后部接口

![CKIPC 后部接口](/assets/image/ckipc/ckipc-rear-interfaces.png)

后部主要接口包括：

1. `DC 24V/1A` 电源输出；
2. RS485 串口接口；
3. LAN 以太网接口，同时作为 IPC 连接口和 PoE 输出口；
4. PSE PoE 电源输入测试口；
5. `DC 12V/3A` 电源输出；
6. LED 白光照明灯；
7. 电池后盖、支架和扬声器。

### 侧面接口

![CKIPC 侧面接口](/assets/image/ckipc/ckipc-side-interfaces.png)

侧面主要接口包括：

1. `DC 12V/2A` 充电接口；
2. USB `5V/2A` 输入/输出接口；
3. 音频输入、音频输出接口；
4. UTP 寻线接口；
5. MicroSD 卡槽；
6. HDMI 高清输入接口；
7. TDR 断点测试接口。

### 顶部和正面接口

- CVBS 模拟视频输入接口；
- 彩条测试信号输出接口；
- 红光源接口；
- 光功率计接口；
- 实体按键控制区；
- 电源指示灯；
- 充电指示灯。

::: danger 接口保护
- LAN、UTP、TDR 和电源接口不得短路；
- TDR 接口不得连接交换机或其他带电网络设备；
- 测线端口不得连接 PoE 交换机；
- 不确定线路是否带电时，请先使用专业设备确认。
:::

4. 进入系统桌面。

## IPC 网络摄像机 IP 探测

IP 探测可用于：

- 自动扫描局域网摄像机；
- 识别摄像机 IP；
- 自动调整测试仪网段；
- PING 摄像机；
- 跳转 ONVIF 预览；
- 部分摄像机激活和测试报告生成。

### 硬件连接

普通供电摄像机：

1. 摄像机网线接入仪器 LAN 口；
2. 摄像机另外连接电源；
3. 打开 IP 探测。

PoE 摄像机：

1. 摄像机网线接入仪器 LAN 口；
2. 确认摄像机支持标准 PoE；
3. 在电源管理中开启 PoE 输出；
4. 再进行 IP 探测。

![CKIPC IP 探测](/assets/image/ckipc/ckipc-ip-detect.png)

## ONVIF 摄像机测试

极速 ONVIF 可自动扫描同一局域网内的 ONVIF 摄像机。

基本步骤：

1. 打开“极速 ONVIF”；
2. 等待扫描完成；
3. 在设备列表中选择摄像机；
4. 输入用户名和密码；
5. 登录并预览实时视频；
6. 根据需要查看设备信息、修改 IP、调整图像或云台参数。

![CKIPC ONVIF 实时预览](/assets/image/ckipc/ckipc-onvif.png)

如果无法发现摄像机，请检查：

- 摄像机是否已经启动；
- 网线是否正常；
- 测试仪和摄像机是否同网段；
- 摄像机是否开启 ONVIF；
- 用户名和密码是否正确；
- PoE 供电是否正常。

## 网络工具

网络工具包含：

- IP 地址扫描；
- PING；
- DHCP 服务；
- 网口闪烁；
- 路由和网络状态检查。

![CKIPC 网络工具](/assets/image/ckipc/ckipc-network-tools.png)

### IP 扫描

输入起始 IP 和结束 IP，扫描指定网段内的设备。发现摄像机后，可跳转至 ONVIF 预览。

### PING

输入目标 IP，可测试：

- 网络连通性；
- 延迟；
- 丢包情况；
- 设备响应状态。

### 网口闪烁

将网线接入 LAN 口，启动网口闪烁，可在交换机端通过规律闪烁的端口灯寻找对应端口。

### DHCP

开启 DHCP 服务后，可为设置为自动获取地址的摄像机临时分配 IP。

## PSE PoE 电压测试

PSE 测试可显示 PoE 交换机端口的供电电压和供电线序。

![CKIPC PSE 电压测试](/assets/image/ckipc/ckipc-pse.png)

基本连接方式：

1. 使用网线连接 PoE 交换机和仪器 `PSE IN` 接口；
2. 测试时不要开启仪器自身的 PoE 输出；
3. 打开 PSE 测试应用；
4. 查看各芯线电压和供电线序。

部分测试场景可通过第二根网线连接 LAN 口和 PoE 摄像机，以验证带载供电。

## 电源输出管理

![CKIPC 电源管理](/assets/image/ckipc/ckipc-power-management.png)

设备可提供：

- USB `5V/2A` 输出；
- `12V/3A` 输出；
- `24V/1A` 输出；
- 标准 `48V` PoE 输出。

::: warning 供电前确认
开启供电前必须确认摄像机的电压、极性和 PoE 标准。错误供电可能造成摄像机或测试仪损坏。
:::

### 48V PoE 输出

1. 将支持标准 PoE 的摄像机连接至 LAN 口；
2. 进入电源管理；
3. 开启 48V PoE；
4. 再进行网络摄像机测试。

### 12V / 24V 输出

使用专用输出线连接摄像机，注意正负极，不要反接。

## RS485 串口工具

RS485 工具可用于：

- 接收和发送串口数据；
- 字符或十六进制显示；
- 设置波特率、数据位、校验位和停止位；
- 分析云台协议和其他串行通信。

连接时，将 RS485 线接入仪器接口，鳄鱼夹或接线端连接被测设备。

## 文件管理和数据导出

文件管理支持：

- 复制；
- 移动；
- 删除；
- 重命名；
- 内部存储与 MicroSD 卡管理；
- U 盘读写；
- FTP 远程文件管理。

仪器接入局域网后，可开启 FTP 服务，然后在电脑文件管理器中输入仪器显示的 FTP 地址，对测试图片、录像和报告进行复制。

## 寻线功能

寻线器支持：

- 普通网线寻线；
- PoE 交换机寻线；
- 同轴线寻线；
- 双芯线缆寻线；
- 灵敏度调节；
- 带载寻线。

标称最大寻线距离可达约 1000 米，但实际距离取决于线缆类型、屏蔽、接地和环境干扰。

::: danger
不得使用本设备寻找交流市电线路。确认线路无 220V 等危险电压后，才能进行寻线。
:::

## 网线通断和线序测试

将网线一端接入仪器 TDR 测试口，另一端连接寻线器测试口或配套远端测试盒。

![CKIPC 网线测试](/assets/image/ckipc/ckipc-cable-test.png)

可检查：

- 1-8 芯通断；
- 线序错误；
- 开路；
- 短路；
- 交叉；
- 故障位于近端或远端；
- 测试报告。

## TDR 断点和线长测试

TDR 断点测试可用于：

- 网线；
- BNC 视频线；
- 电话线；
- TVVB 等线缆；
- 自定义线缆。

标称最大测试距离约 1000 米。

![CKIPC TDR 测试](/assets/image/ckipc/ckipc-tdr.png)

普通网线测试时：

1. 网线接入 TDR 口；
2. 另一端保持悬空；
3. 打开 TDR 测试；
4. 选择线缆类型；
5. 查看长度或断点距离。

其他线缆可通过 RJ45 转 BNC 或鳄鱼夹转接。

## 网线 TDR 综合测试

![CKIPC 网线 TDR 综合测试](/assets/image/ckipc/ckipc-cable-tdr.png)

可显示：

- 网线长度；
- 衰减；
- 质量；
- 反射率；
- 阻抗；
- 线对状态。

说明书标注的长度测试范围约为 `2-200 m`，精度约 `0.8 m`。衰减测试通常要求线缆长度达到一定范围。

## 光功率计

光功率计支持以下波长校正点：

- `850 nm`
- `1300 nm`
- `1310 nm`
- `1490 nm`
- `1550 nm`
- `1625 nm`

![CKIPC 光功率计](/assets/image/ckipc/ckipc-opm.png)

操作步骤：

1. 清洁光纤连接器和仪器接口；
2. 将光纤接入光功率计接口；
3. 打开光功率计应用；
4. 选择对应波长；
5. 读取 dBm 或线性功率；
6. 根据需要保存测试结果。

## 红光故障定位

红光源输出波长约为 `650 nm`，支持：

- 常亮；
- 1 Hz 闪烁；
- 2 Hz 闪烁。

![CKIPC 红光源](/assets/image/ckipc/ckipc-vfl.png)

红光源可辅助查找：

- 光纤外皮破损；
- 跳线连接不良；
- 光纤严重弯曲；
- 近距离断点。

标称可视测试距离最长可达约 1000 米，实际效果受光纤类型、环境光和故障情况影响。

::: danger 激光安全
红光源开启后，不得直视仪器接口、跳线端面或光纤端面。
:::

## 充电和电池维护

- 使用 `DC 12V/2A` 充电器；
- 建议关机或低负载状态下充电；
- 不建议长期边充电边进行大功率 PoE 输出；
- 长期不用时应定期补充电量；
- 避免高温、低温和潮湿环境；
- 电池异常发热、膨胀或续航明显下降时，应停止使用并联系售后。

## 常见问题

### 摄像机无法显示图像

检查：

- 视频接口是否连接正确；
- 摄像机制式是否受支持；
- 摄像机是否正常供电；
- BNC 或网线是否正常；
- IPC 地址、网段、用户名和密码是否正确；
- ONVIF 是否开启。

### PoE 摄像机无法启动

检查：

- 摄像机是否支持标准 PoE；
- PoE 输出开关是否开启；
- 网线线序和水晶头是否正常；
- 仪器电池电量是否充足；
- 摄像机功耗是否超过仪器输出能力。

### IP 扫描不到摄像机

检查：

- LAN 口连接；
- 摄像机启动状态；
- IP 地址范围；
- DHCP 或静态 IP 配置；
- Wi-Fi 与有线网络是否冲突；
- 摄像机是否处于未激活状态。

### TDR 结果不稳定

检查：

- 线缆类型是否选择正确；
- 另一端是否按要求悬空；
- 转接头是否可靠；
- 线缆是否存在分支；
- 是否需要进行手动校准。

## 使用与维护注意事项

- 设备应存放在干燥、无腐蚀性气体的环境中；
- 避免雨淋、受潮、跌落和剧烈振动；
- 不要擅自拆机或修改系统；
- 定期清洁接口和防护胶套；
- 光纤接口未使用时应盖好防尘帽；
- 插拔网线和 BNC 接头时不要用力拉扯；
- 升级系统前应保证电量充足；
- 测试报告、照片和录像应定期备份。
