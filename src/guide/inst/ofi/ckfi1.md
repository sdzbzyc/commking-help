# CKFI-1 光纤识别仪使用说明

CKFI-1 光纤识别仪用于在**不中断现有通信业务**的情况下识别目标光纤，并判断光信号方向和强弱。仪器适用于单模及多模光纤，可识别 `270 Hz`、`1 kHz` 和 `2 kHz` 调制信号，常用于光纤维护、安装、布线和故障恢复。

::: warning 激光安全
不要直视光纤网络设备、测试设备、跳线或测试跳线的光输出端，以免造成眼睛伤害。
:::

## 产品特点

- 数字显示相对功率；
- 在线测试，无需中断业务；
- 识别 `270 Hz`、`1 kHz`、`2 kHz` 调制信号；
- 一键式操作，使用简单；
- 配有遮光罩，有助于提高测试准确性；
- 配有多种夹具，可适应不同规格的裸纤和尾纤；
- 采用机械阻尼设计，降低夹持过程中损伤光纤的风险；
- 可通过左右方向指示灯判断信号传输方向。

## 夹具选择与安装

仪器配有适用于不同光纤规格的夹具。请根据被测光纤选择正确的夹具，避免因夹具不匹配造成测试误差或光纤损伤。

支持的光纤及尾纤规格包括：

- `φ0.25 mm` 裸纤；
- `φ0.9 mm` 尾纤；
- `φ2.0 mm` 尾纤；
- `φ3.0 mm` 尾纤。

![CKFI-1 夹具安装方法](/assets/image/ckfi1/ckfi1-fixtures.png)

安装步骤：

1. 将夹持按钮向下推至最大位置；
2. 选择与待测光纤匹配的夹具；
3. 将夹具放置在显示屏上方的安装位置；
4. 从仪器右侧将夹具推入并安装到位；
5. 确认夹具安装牢固后再放入光纤。

::: caution
请勿将光纤强行压入夹具或使光纤与光纤槽错位，否则可能造成光学组件变形或损伤光纤。
:::

## 光纤信号识别

![CKFI-1 光纤信号识别方法](/assets/image/ckfi1/ckfi1-identification.png)

操作步骤：

1. 将待测光纤轻轻放入夹具凹槽；
2. 推动夹持按钮锁紧夹具；
3. 安装遮光罩；
4. 读取显示屏上的功率、频率和方向信息。

测试结果说明：

- **光纤中存在信号**：左右方向指示灯显示信号方向，显示屏显示信号功率；
- **没有检测到有效信号**：显示屏可能显示 `LO`，方向灯可能左右闪烁；
- **未夹持光纤时**：方向灯也可能出现左右闪烁，属于正常现象；
- **识别到调制信号时**：对应的 `270 Hz`、`1 kHz` 或 `2 kHz` 指示信息显示，并可能伴随蜂鸣声；
- 信号过弱时，仪器可能无法正确识别调制频率。

## 工作原理

使用时，需要在目标光纤的一端注入 `1310 nm` 或 `1550 nm`、带有调制音的测试信号。CKFI-1 在线夹持光纤后，通过检测泄漏光来识别目标光纤，并判断信号方向。

常见调制频率：

- `270 Hz`
- `1 kHz`
- `2 kHz`

## 使用注意事项

![CKFI-1 使用注意事项](/assets/image/ckfi1/ckfi1-safety.png)

1. 使用前请阅读全部说明和警告；
2. 不要在高温、强光、强磁场或靠近火源的环境中存放仪器；
3. 保持设备和夹具清洁，避免污垢和污染物影响测试；
4. 将光纤轻放入夹具，不要强压或弯折光纤；
5. 不要直视任何光输出端；
6. 避免敲击、投掷、踩踏、挤压等强烈物理作用；
7. 非专业人员不得拆解仪器；
8. 使用不当造成的人身或财产损失，由使用者自行承担。

## 常见问题

### 显示 `LO` 是否表示仪器损坏？

不一定。`LO` 通常表示没有检测到有效光信号或信号功率过低。请检查：

- 光纤是否正确放入夹具；
- 夹具规格是否与光纤匹配；
- 遮光罩是否安装；
- 光纤中是否注入有效测试信号；
- 测试信号功率是否在仪器检测范围内。

### 方向指示灯左右闪烁是否正常？

在没有夹持光纤、光纤中没有有效信号或信号过弱时，方向指示灯可能左右闪烁。

### 为什么不能识别调制频率？

可能原因包括：

- 测试信号不属于 `270 Hz`、`1 kHz`、`2 kHz`；
- 信号功率过低；
- 夹具选择错误；
- 光纤未正确放入光纤槽；
- 外界光线干扰较强，未安装遮光罩。

## 维护与存放

- 使用后清洁光纤槽和夹具；
- 长期不用时取出电池，避免漏液；
- 将主机和夹具存放在干燥、清洁的环境中；
- 避免高温、强光、强磁场及潮湿环境；
- 仪器异常时请联系售后人员处理，不要自行拆机。
