## 一、介绍
感谢您购买本公司光纤熔接机`CK100`

`CK100`可以通过装配的显微镜自动进行单芯光纤熔接前的检查和对轴，是低接续损耗的光纤熔接设备。此外，还可以对保护接续点的保护套管进行加热和收缩。

使用` CK100` 之前，本公司建议您认证阅读本使用说明书，
确保正确有效地使用 `CK100` 光纤熔接机。

## 二、可接续的光纤种类

| 材质         | 石英玻璃                                                     |
| ------------ | ------------------------------------------------------------ |
| 光纤种类     | SMF(ITU-T G.652) 单模光纤<br>MMF(ITU-T G.651) 多模光纤<br>DSF(ITU-T G.653) 色散位移光纤<br>NZDSF(ITU-T G.655)非零色散偏移光纤<br>BIF(ITU-T G6.57) 弯曲不敏感单模光纤<br/>EDF掺铒光纤 |
| 光纤外径     | 125μm                                                        |
| 光纤涂敷外径 | 900μm                                                        |
| 光纤芯数     | 单芯                                                         |
| 切断长度     | 10-16mm                                                      |

CK100 不仅有单模熔接模式，也有多模模式，AUTO 模式。

AUTO 模式包含[SM AUTO], [MM  AUTO], [NZ AUTO] , [DS AUTO] 模式。

当使用 AUTO 模式时，自动电弧校准功能会主动激活。 

标准模式包含 [SM] 适用于 SMF (ITU-T G652)光纤, [NZ] 适用于 NZDSF (ITU-T G655)光 纤, [DS]适用于 DSF (ITU-T G653)光纤。 

以及 [MM] 适用于 MMF (ITU-T G651)光纤。该模式在熔接前需要执行[放电校正]操作。

## 电源操作

本章节将指导如何使用电源。

熔接机由AC适配器或内部的电池供电工作。*`CK100`只能使用专门的外部适配器和AC电源线。*

:::danger 警告
CK100熔接机的原装适配器输入参数为13.5V，5A，使用非原装适配器会使机器短路，此行为为人为损坏，不在质保范围内，会造成高额的维修费用。
:::
### 安装电池

请将电池（BTR-72）安装在 CK100 底部的电池槽内，直到它正确到位。 安装和取出的方法参考如下

![电池](/assets/image/ck100/ck100_5.jpg)

### 电池充电

适配器 ADC-135 连接到接口后，电池 BTR-72 开始充电。充电指示 LED 灯（红色）闪烁并进入 充电状态，之后将花费 4 小时来充电并熄灭 LED，在充电完成后拔出 AC 电源线插头。 在充电时能够进行熔接操作。

### 清洁光纤

用蘸有酒精的纱布或足够软的纸巾清洁从尾部算起大约500mm左右的光纤，如果光纤涂覆层表面的灰尘微粒进入热缩套管内部，将会引起光纤的断裂或者损耗增大。

### 将光纤传入热缩套管

从光纤右边穿入热缩套管。

:::tip 提示
推荐使用右手拿住热缩套管，然后左手拿住光纤沿图片指示的方向穿入。
:::
