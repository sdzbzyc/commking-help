# CK6105 光源万用表使用说明

CK6105 光源万用表集 **稳定光源、光功率计、可视红光源（选配）** 于一体，主要用于连续光信号功率测量、光纤线路通断测试、光纤线路故障排查及光纤线路损耗测试，适用于光纤施工与维护、光纤通信、光纤传感、光 CATV 等场景。

> 本页面根据产品说明书整理。说明书版本如有变更，请以实际随货资料或厂家最新资料为准。

## 产品特点

- 支持自动关机。
- 可选配大容量锂电池，可通过 USB 充电口充电。
- 可选配高功率可视红光。
- 光源支持 CW 和多种调制光输出。
- 光源输出功率可调，调节范围 6 dB，步进 1 dB。
- 光源支持组合波长定制，例如 1310 / 1490 / 1550 nm、850 / 1300 nm，也可根据需求定制 1625 nm、1650 nm 等波长。
- 光功率计支持线性 mW 和非线性 dBm 同时显示。
- 光功率计支持 850 / 980 / 1300 / 1310 / 1490 / 1550 / 1625 / 1650 nm 八个测量波长。
- 光功率计支持波长记忆和参考功率记忆。

## 按键说明

::: tip 操作说明
短按：按下时间小于 1 秒。  
长按：按下时间大于 2 秒。
:::

### 1. 电源键

- 关机状态下，短按开机。
- 开机状态下，短按开启或关闭定时关机功能。
- 定时关机功能开启后，无操作 10 分钟后自动关机。
- 开机状态下，长按关机。

### 2. LIGHT 键

- 短按开启或关闭背光功能。
- 可视红光功能：长按开启可视故障定位功能。
- 在可视红光功能开启后，再次按此键可切换 **常亮、闪烁、关闭**。

### 3. ▲ / REF 键

- 稳定光源：按下此键调节衰减。
- 光万用表 - 光源功能：按下此键调节衰减或调制频率。
- 光万用表 - 光功率计功能：把当前光功率设置为参考功率，并保存。

### 4. ▼ / dB 键

- 稳定光源：按下此键调节衰减。
- 光万用表 - 光源功能：按下此键调节调制频率。
- 光万用表 - 光功率计功能：开启或关闭相对光功率测量，用于测量链路损耗。

### 5. MOD 键

- 稳定光源：切换光源调制频率。
- 光万用表：切换稳定光源功能和光功率计功能。

### 6. λ 波长切换键

- 稳定光源：切换输出光源波长。
- 光万用表：切换光源或功率计波长。

## 光功率计校准

### 用户校准

1. 同时按下 **MOD** 和 **LIGHT** 键，进入校准模式。
2. 进入校准模式后，屏幕右下角显示字符 **K**。
3. 按下 **▲ / REF** 键，每次增加 0.05 dB。
4. 按下 **▼ / dB** 键，每次减小 0.05 dB。
5. 调整完成后，按电源键保存校准数据。

::: warning 注意
用户校准会影响测量结果。非必要情况下，不建议随意调整校准值。
:::

### 恢复出厂设置

同时按下 **▲ / REF** 和 **▼ / dB** 键，可恢复出厂设置。

## 常见故障解决办法

| 故障提示 | 可能原因 | 解决方法 |
|---|---|---|
| LCD 显示微弱 | 电源不足 | 更换电池或及时充电 |
| 开机无法显示 | 电源不足或其他 | 重新开机或及时充电 |
| LCD 显示数据不正常 | 接头故障、污浊或被锁定 | 重新连接接头及清洁探测器 |
| 光源输出功率弱 | 光纤接头不清洁 | 拆下 FC 接口，清洁端面 |

## 日常保养与维护

1. 请经常保持光纤端面清洁，做到无油脂、无污染。不要使用不清洁或非标准适配器接头，也不要插入抛光面差的端面，否则可能损坏光纤端面并造成测试误差。
2. 尽可能坚持使用同一种适配器。
3. 当仪表不使用时，请立即盖上防尘帽，保护端面清洁，避免长时间暴露在空气中附着灰尘而产生测量误差。
4. 小心插拔光适配器接头，避免端口造成刮痕。
5. 定期清洁传感器表面。清洁传感器表面时，请使用专用清洁棉签，沿圆周方向轻轻擦拭。
