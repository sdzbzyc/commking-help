# CKOT-2 系列光时域反射仪使用说明

CKOT-2 系列光时域反射仪是一款多功能光纤测试仪表。根据具体型号，仪器可集成 OTDR、事件地图、光功率计、可视故障定位、稳定光源、RJ45 线序、RJ45 线长与寻线等功能，适用于光纤链路施工、维护、故障定位和测试结果分析。

> [!DANGER]
> 使用仪器时，严禁用眼睛直视激光输出口或光纤尾端，否则可能损伤视网膜，严重时可导致失明。

## 一、安全与使用注意事项

- 不要将设备暴露在雷雨或潮湿环境中，以降低火灾和触电风险。
- 不要自行打开仪器外壳，维修应由厂家指定的专业人员完成。
- 机内使用专用锂离子聚合物电池，充电电压为 `5V`。
- 电池充电温度范围为 `0℃～50℃`；环境温度过高时，充电会自动终止。
- 长期存放温度范围为 `-20℃～45℃`。
- 请使用随仪器提供的专用 AC 适配器，并按规范连接外部电源。
- 仪器采用 `3.5 英寸彩色 LCD`。清洁屏幕时，请使用柔软织物轻轻擦拭。

## 二、主机接口与部件

### 顶部接口

| 编号 | 接口 | 用途 |
|---|---|---|
| 1 | OTDR / LS 光接口 | 用于 OTDR 测试或稳定光源输出 |
| 2 | OPM 光功率计接口 | 用于光功率测量 |
| 3 | VFL 可视故障定位接口 | 输出 650nm 红光进行近端故障定位 |
| 4 | 手电筒 | 提供现场照明 |

### 左侧接口

| 接口 | 用途 |
|---|---|
| Micro USB | 充电、连接电脑及拷贝数据 |
| LED 充电指示灯 | 显示充电状态 |
| TF（Micro SD）卡插槽 | 存储测试数据 |

### 右侧及底部接口

| 接口 | 用途 |
|---|---|
| RJ45 测试接口 | 进行线序、线长或寻线测试 |
| 复位接口 | 仪器异常时执行复位 |
| RJ45 测试远端 | 与主机配合完成网线测试 |

![CKOT-2 主机接口与部件](/assets/image/ckot2/ckot2-02-host-and-ports.png)

## 三、按键说明与主界面

### 按键说明

| 按键 | 功能 |
|---|---|
| F1～F4 | 执行当前界面底部对应的功能 |
| 缩放键 | 在 OTDR 曲线界面进入或退出缩放模式 |
| 测量键 | 在 OTDR 界面启动或停止测试 |
| 电源键 | 短按开机，长按关机；开机后短按可打开或关闭手电筒 |
| 返回键 | 返回上一级菜单 |
| 方向键 | 上下左右选择菜单、参数或光标位置 |
| OK 键 | 进入下一级页面或确认操作 |

### 主界面功能

仪器开机后进入主菜单。根据具体型号，主菜单可能显示 6 项或 8 项功能。使用方向键选择功能，再按 `OK` 键进入。

常见功能包括：

- OTDR 光时域反射仪
- 光眼（事件地图）
- 光功率计 OPM
- VFL 可视故障定位
- 稳定光源
- RJ45 线序
- RJ45 线长与寻线
- 系统设置

> 部分功能仅在相应型号或选配版本中提供。

![CKOT-2 功能按键与主界面](/assets/image/ckot2/ckot2-03-buttons-and-main-menu.png)

## 四、OTDR 光时域反射测试

### OTDR 主界面

OTDR 主界面通常包含以下区域：

- 链路信息
- 波形显示区域
- A/B 光标及其测量信息
- 测试条件
- 事件列表
- 当前功能菜单

底部功能键对应关系如下：

| 按键 | 功能 |
|---|---|
| F1 | 进入参数设置界面 |
| F2 | 切换 A/B 光标 |
| F3 | 进入保存界面 |
| F4 | 打开已存储的曲线文件 |

### 设置测量参数

在 OTDR 设置界面中，可以配置：

- 测量波长
- 测量模式
- 测量范围
- 脉冲宽度
- 测量时间
- 反射阈值
- 事件损耗阈值
- 结束门限
- 光纤折射率

多位数参数的设置方法：使用左右方向键移动光标，再使用上下方向键调整数值。

| 按键 | 设置界面功能 |
|---|---|
| ▲ / ▼ | 切换设置条目 |
| OK | 确认或编辑当前参数 |
| F1 | 启动测试 |
| F2 | 确认参数设置 |
| F3 | 恢复出厂设置 |
| F4 | 取消并返回 |

![CKOT-2 OTDR 主界面与参数设置](/assets/image/ckot2/ckot2-04-otdr-overview-settings.png)

### 启动测试与查看结果

1. 清洁被测光纤端面并连接到 OTDR 接口。
2. 进入 OTDR 功能。
3. 按 `F1` 进入参数设置，设置波长、量程、脉宽等参数。
4. 按测量键或设置界面的 `F1` 启动测试。
5. 测试完成后，在屏幕上方查看链路质量和链路信息。
6. 通过事件列表查看每个事件的详细数据。

链路信息通常包括：

- 链路长度
- 链路总损耗
- 平均损耗
- 事件点数量
- A/B 光标之间的距离与损耗

事件类型主要包括：

1. 反射事件
2. 下降事件
3. 分路器事件
4. 光纤末端

### 波形缩放

在 OTDR 曲线界面按缩放键进入缩放模式，再配合方向键执行以下操作：

- X 轴方向放大或缩小
- Y 轴方向放大或缩小
- 按 `OK` 恢复为 `1:1` 显示
- 再次按缩放键退出缩放模式

![CKOT-2 OTDR 测试结果与波形缩放](/assets/image/ckot2/ckot2-05-test-result-and-zoom.png)

## 五、测试文件保存与打开

### 保存测试文件

1. 测量结束后按 `F3（存储）`。
2. 在弹出的键盘中输入文件名称。
3. 选择 `Enter` 保存文件。

也可以在系统设置中启用 **OTDR 自动保存**。启用后，平均测试或自动测试结束时，仪器会自动保存测试文件。

### 打开已保存文件

1. 在 OTDR 界面按 `F4（文件）` 进入文件列表。
2. 使用方向键选择文件夹或文件。
3. 按 `OK` 打开。

| 按键 | 文件列表功能 |
|---|---|
| F1 | 打开文件 |
| F2 | 删除文件 |
| F3 | 重命名文件 |
| F4 | 返回主菜单 |

![CKOT-2 测试文件保存与打开](/assets/image/ckot2/ckot2-06-file-save-open.png)

## 六、光眼（事件地图）

光眼功能可对被测光纤链路执行一键自动测试，并以图形化方式显示：

- 链路总长度
- 链路损耗
- 平均损耗
- 回波损耗
- 事件点类型
- 事件位置
- 链路质量 PASS / FAIL

使用左右方向键可以切换当前事件，并查看该事件的详细信息。

## 七、光功率计 OPM

光功率计用于光信号功率、插入损耗等测试，并可识别和测量 `270Hz`、`330Hz`、`1kHz`、`2kHz` 调制光信号。

| 按键 | 功能 |
|---|---|
| F1 | 切换测量波长 |
| F2 | 设置参考功率 |
| F3 | 参考功率置零 |
| F4 | 进入校准模式 |

界面可显示：

- 绝对功率（dBm）
- 相对功率（dB）
- 线性功率（mW）
- 当前测量波长
- 参考功率
- 所测光信号频率

功率换算关系：

```text
绝对功率（dBm）= 10 × lg（线性功率 / 1mW）
相对功率（dB）= 绝对功率 - 参考功率
```

![CKOT-2 光眼与光功率计](/assets/image/ckot2/ckot2-07-event-map-and-opm.png)

## 八、VFL 可视故障定位

VFL 将 `650nm` 可视红光注入光纤，通过观察漏光位置，帮助判断近端故障点或微弯造成的高损耗区段。

适用于：

- 裸光纤
- 光纤跳线
- 可泄漏红光的光纤或光缆
- 近端断点和高损耗区段检查

| 按键 | 功能 |
|---|---|
| F1 | 打开红光，常亮模式 |
| F2 | 红光以 1Hz 频率闪烁 |
| F3 | 红光以 2Hz 频率闪烁 |
| F4 | 关闭红光 |

> [!DANGER]
> 不要直视 VFL 红光输出口或接入红光后的光纤端面。

## 九、稳定光源 LS

稳定光源的波长通常与 OTDR 波长一致，可用于：

- 电信、CATV、LAN 光缆参数测试
- 光无源器件插入损耗测试
- 隔离度和回波损耗测试
- 探测器波长响应测试

支持的输出模式包括：

- CW
- 270Hz
- 330Hz
- 1kHz
- 2kHz

| 按键 | 功能 |
|---|---|
| F1 | 打开光源 |
| F2 | 关闭光源 |
| F3 | 切换光源波长 |
| F4 | 切换光源模式 |

![CKOT-2 VFL 与稳定光源](/assets/image/ckot2/ckot2-08-vfl-and-light-source.png)

## 十、RJ45 网线测试

> 部分型号提供 RJ45 测试功能。

### RJ45 线序测试

测试前请连接远端模块。

| 按键 | 功能 |
|---|---|
| F1 | 启动线序测试 |
| F3 | 切换线序测试标准 |
| F4 | 返回主菜单 |

仪器会显示 1～8 芯的连通和排列情况，帮助识别断路、错序等问题。

### RJ45 线长与寻线

寻线功能启动后，在远端使用寻线模块查找目标线缆；听到连续提示音时，即可确认被查找线缆。

| 按键 | 功能 |
|---|---|
| F1 | 启动线长测试 |
| F2 | 切换长度单位 |
| F3 | 切换线序测试标准 |
| F4 | 启动寻线功能 |

## 十二、日常维护建议

- 每次连接光纤前，清洁连接器和跳线端面。
- 未使用光接口时，及时盖好防尘帽。
- 不要使用污染、划伤或型号不匹配的光纤接头。
- 长时间不用仪器时，应定期给电池充电。
- 保持 LCD 屏幕清洁，避免用尖锐物体碰撞屏幕。
- 测量结果异常时，先检查接头清洁度、波长、量程、脉宽和折射率等设置。
- 仪器异常且无法通过复位恢复时，请联系售后服务，不要自行拆机维修。
