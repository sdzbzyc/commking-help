# CKOT-1A 光时域反射仪使用说明

CKOT-1A 光时域反射仪主要用于光纤链路测试、故障定位、事件分析、链路损耗测量、测试结果保存与导出等场景。仪器集成 OTDR、光功率计、稳定光源、可视故障定位、事件地图、光损耗测试、端面检测等功能，适用于光纤施工、维护、验收和故障排查。

> 使用仪器前，请先阅读安全提示，并按说明正确连接光纤、设置测试条件和保存测试结果。测试过程中请勿直视激光输出口、红光输出口或光纤尾端。

## 一、安全提示

### 外部电源

- 电源适配器输入：`100V～240V，50/60Hz`
- 电源适配器输出：`19V±1V，1.31A`
- 极性要求：中心为正

请严格按照规范使用外部电源，否则可能造成设备损坏。

### 内部电池

- 仪器内部为专用锂电池。
- 首次使用时，建议先使用内部电池供电，将电池电量耗尽后再充电。
- 首次充电时间建议不少于 `10 小时`。
- 充电温度范围：`0℃～50℃`。
- 仪器长期闲置超过 `2 个月` 时，应及时充电以保持电池电量。
- 长期存放时，建议取出电池。
- 电池长期存放温度范围：`-20℃～45℃`。

### 激光安全

- 使用仪器时，严禁眼睛直视激光输出口。
- 测试时不要直视光纤尾端。
- 可视红光功能开启时，不要用眼睛直视红光输出端。
- 仪器使用完毕后，请盖好光输出口防尘帽。

## 四、使用准备

### 安装电池

1. 拧下电池盖上的螺钉。
2. 取下电池盖，装入电池。
3. 盖上电池盖并拧紧螺钉。

![电池安装示意](/assets/image/ckot1/ckot1-03-battery-install.jpg)

### 开机与电量提示

1. 按下仪器前面板的电源键。
2. 仪器正常启动后，`ON` 指示灯点亮。
3. 屏幕显示主菜单。

如果使用 AC 适配器而非电池供电，`CHARGE LED` 不会点亮。电池电量低时，屏幕会出现警告信息，请及时连接适配器充电。

### 连接光纤

连接光纤前，请先确认：

- 接头类型是否匹配；
- 接头端面是否清洁；
- 光纤端面没有明显污染、划伤或损坏；
- 不要在光源输出期间直视端口或光纤尾端。

正确清洁端面的方法是：在连接跳线前，使用无水酒精清洁跳线端面，待酒精挥发后再连接仪器。

![开机、电量提示与连接光纤注意事项](/assets/image/ckot1/ckot1-04-power-fiber-connect.jpg)

## 五、OTDR 测量

### 1. 设置测量条件

在 OTDR 测量界面按 `设置` 键，进入测量条件设置界面。常见参数包括：

| 参数 | 说明 |
|---|---|
| 测量波长 | 根据测试链路选择 1310nm、1550nm 等波长 |
| 测量模式 | 自动测试、实时测试、平均测试 |
| 测量范围 | 根据光纤长度选择合适量程 |
| 脉冲宽度 | 根据链路长度和分辨率要求选择 |
| 平均次数 | 用于平均测试，提高曲线稳定性 |
| 折射率 | 根据光纤类型设置 |
| 光缆修正 | 用于修正光缆长度 |
| 熔接门限 | 设置熔接事件判断门限 |
| 反射门限 | 设置反射事件判断门限 |
| 结束门限 | 设置链路结束判断门限 |
| 锁定 A/B 光标 | 控制 A/B 光标是否锁定 |

### 2. 自动测量

自动测量适合快速测试。

操作步骤：

1. 进入 OTDR 界面。
2. 按 `设置` 键。
3. 选择测试波长。
4. 选择测量模式为 `自动测试`。
5. 选择测量范围。
6. 开始自动测量。

![OTDR 自动测量设置界面](/assets/image/ckot1/ckot1-05-auto-measure-settings.jpg)

### 3. 手动测量

手动测量适合专业测试，测试人员可根据光缆实际情况设置测试条件。

操作步骤：

1. 按 `设置` 键。
2. 选择测量模式。
3. 选择 `平均测试`。
4. 设置合适的测试参数，提高测试结果准确性。
5. 开始测量。

![OTDR 手动/平均测量设置界面](/assets/image/ckot1/ckot1-06-manual-measure-settings.jpg)

### 4. 实时测量

实时测量适合观察链路变化和快速判断接续状态。

操作步骤：

1. 按 `设置` 键。
2. 选择测试波长。
3. 选择测量模式为 `实时测试`。
4. 选择测量范围。
5. 选择脉冲宽度。
6. 开始实时测量。

![OTDR 实时测量设置界面](/assets/image/ckot1/ckot1-07-realtime-measure-settings.jpg)

### 5. 启动测量

进入 OTDR 测试界面后：

1. 按 OTDR 测试界面右侧菜单中的 `测试` 键，开始测量。
2. 也可以按仪器按键区的 `REAL / AVG`，按上一次设置启动 `实时` 或 `平均` 测量。

## 六、查看测试结果

### 曲线分析结果

测试完成后，仪器界面会显示测试结果、曲线、事件操作区和事件列表。选择事件列表中的事件后，曲线上对应事件会被标注。

![OTDR 测试结果与事件列表](/assets/image/ckot1/ckot1-08-test-result-event-list.jpg)

| 区域 | 说明 |
|---|---|
| 测试结果 | 显示损耗、平均损耗、距离等结果 |
| 曲线显示 | 显示 OTDR 测试曲线 |
| 曲线操作 | 用于缩放、移动、A/B 光标切换 |
| 事件列表 | 显示链路中的事件点 |

### 测量两点之间距离和平均损耗

1. 点击曲线目标位置，当前选中的光标会跳转到目标位置。
2. 使用方向键或菜单按钮移动光标。
3. 使用 `A/B` 键切换 A、B 光标。
4. 屏幕上方会显示 A-B 两点之间的距离、损耗和平均损耗。

![A/B 两点距离、损耗和平均损耗测量](/assets/image/ckot1/ckot1-09-ab-distance-loss.jpg)

| 垂直缩小 | 缩小纵向显示 |
| 水平放大 | 放大横向显示 |
| 水平压缩 | 压缩横向显示 |

| A/B 切换 | 切换 A、B 光标 |
| 向右移动 | 右移光标或曲线 |

## 七、测试结果保存与导出

### 保存测试结果

在 OTDR 主界面中：

1. 选择 `文件` 菜单。
2. 在弹出菜单中选择 `保存`。
3. 输入文件名、保存路径。
4. 按确认菜单保存。

界面中的 `一键存储` 为快捷保存模式，勾选后可快速保存测试文件。

![OTDR 测试结果保存界面](/assets/image/ckot1/ckot1-11-save-result.jpg)

### 导出测试结果

在 OTDR 主界面中：

1. 选择 `文件` 菜单。
2. 在弹出菜单中选择 `拷贝`。
3. 进入文件拷贝界面。
4. 可选择拷贝单个文件或全部文件。
5. 选择目标 U 盘文件位置后，点击拷贝图标完成导出。

![OTDR 测试文件导出界面](/assets/image/ckot1/ckot1-12-export-result.jpg)

## 九、光源功能

光源功能用于输出稳定光信号。

![光源功能界面](/assets/image/ckot1/ckot1-14-light-source.jpg)

操作要点：

1. 进入 `光源` 功能界面。
2. 选择输出波长，例如 `1310nm`、`1550nm` 等。
3. 根据需要开启或关闭对应波长输出。
4. 输出口通常为仪器顶部的光源端口。

## 十、VFL 可视故障定位

VFL 可视故障定位功能用于通过红光观察近距离断点、弯折点、接续异常点等问题。

![VFL 可视故障定位界面](/assets/image/ckot1/ckot1-15-vfl.jpg)

常见模式：

| 模式 | 说明 |
|---|---|
| 常亮 | 红光持续输出 |
| 1Hz | 低频闪烁 |
| 2Hz | 高频闪烁 |
| 关闭 | 关闭红光输出 |

> VFL 红光输出时，请勿直视红光端口或光纤尾端。

## 十一、光损耗测试

光损耗测试为选配功能，用于配合光源和光功率计进行链路损耗测量。

![光损耗测试界面](/assets/image/ckot1/ckot1-16-loss-test.jpg)

基本流程：

1. 将标准跳线、被测件按界面提示连接到 `OPM` 与 `LS` 端口。
2. 设置光源波长。
3. 设置光功率计测量波长。
4. 设置功率计参考值。
5. 选择功率单位。
6. 读取插入损耗、绝对功率等测试结果。

## 十二、端面检测

端面检测为选配功能，用于检查光纤端面污染、划伤、缺陷等问题。

使用时应根据仪器配置连接端面检测探头，并按界面提示进行观察和判断。

## 十四、仪器维护

### 连接器清洁

在仪器使用过程中，应保持端面清洁。若仪器无法测试出正常曲线，或测试结果不准确，应优先检查连接器是否清洁。

清洁要求：

1. 清洁时请确认 OTDR 和 VFL 功能处于停止状态。
2. 按下输出接口。
3. 使用经过酒精润湿的专用纸巾或棉签擦拭连接端面。
4. 使用完毕后盖好防尘帽。
5. 仪器正常使用过程中也应保持防尘帽清洁。

### 内部电池维护

- 首次使用时，建议先使用电池供电至彻底放电，再充电。
- 首次充电时间建议大于 `8 小时`。
- 充电温度范围为 `0℃～50℃`。
- 长期不用时，建议每 `2～3 个月` 充电一次，以保持电池性能。

### 屏幕清洁与校准

- 仪器显示屏为 `5.8 英寸彩色触摸 LCD`。
- 不要使用尖锐物体点击液晶屏。
- 清洁时可使用柔软织物和清洁液。
- 不要使用有机溶剂擦拭屏幕。
- 建议 OTDR 每两年校准一次，具体校准事项请联系供应商。

![仪器维护及故障排除说明](/assets/image/ckot1/ckot1-20-maintenance.jpg)

## 十五、故障及处理方法

| 故障描述 | 可能原因 | 处理方法 |
|---|---|---|
| 仪器无法正常启动 | 电池没有电 | 给电池充电并观察开关键灯；若显示红灯则继续充电，否则联系供应商 |
| 仪器无法正常充电 | 使用环境不满足充电条件 | 将仪器置于 `0℃～50℃` 环境中充电 |
| 仪器无法正常充电 | 电池接触不良 | 打开电池仓盖，检查触点并重新安装电池 |
| 仪器无法正常充电 | 电池问题或内部电路问题 | 联系供应商更换电池或检修 |
| 无法测出正常曲线 | 仪器参数设置不正确 | 重新设置正确的测试参数 |
| 无法测出正常曲线 | 光纤输出端面受污染 | 清洁光输出端面 |
| 无法测出正常曲线 | 仪器光输出连接器损坏 | 更换输出连接器 |
| 无法测出正常曲线 | 光输出连接器不匹配 | 更换匹配的连接器 |
| 测试曲线毛刺大、波形不平滑 | 输出接口连接不正确 | 重新连接合适的输出接口 |
| 测试曲线毛刺大、波形不平滑 | 脉宽设置偏小 | 增加测试脉宽值 |
| 测试曲线毛刺大、波形不平滑 | 通道设置不正确 | 修改测试通道参数 |
| 测试曲线前端出现饱和或平顶现象 | 脉宽设置偏大 | 减小测试脉宽参数 |
| 无法测试出光纤末端反射峰 | 量程设置偏小 | 增加测试量程值 |
| 无法测试出光纤末端反射峰 | 脉宽设置偏小 | 增大测试脉宽参数 |
| 曲线分析出现误报 | 测试曲线质量差 | 增大测试脉宽参数 |
| 曲线分析出现误报 | 事件阈值设置偏小 | 增加事件阈值 |
| 测量的光纤长度不准确 | 仪器参数设置不正确 | 重新设置合适的参数 |
| 测量的光纤长度不准确 | 光纤折射率设置不准确 | 重新设置光纤折射率 |
| 测量的光纤长度不准确 | 光缆修正系数设置不准确 | 重新设置光缆修正系数 |
| 测量的光纤平均损耗值不准确 | 测试曲线前端拖尾太长 | 清洁光输出端面 |
| 测量的光纤平均损耗值不准确 | 光标点位置设置不当 | 重置光标点位置 |

> 仪器使用过程中，未经允许严禁用户擅自拆机处理，否则将失去保修资格。

## 十八、使用建议

- 测试前先清洁跳线和端口端面。
- 根据链路长度选择合适量程和脉冲宽度。
- 短距离测试建议使用较小脉宽，长距离测试建议适当增大脉宽。
- 平均测试比实时测试更稳定，适合正式验收。
- 测试完成后及时保存 `.SOR` 文件。
- 导出测试数据前请确认 U 盘已正确连接。
- 长期不用时，请定期充电并保持端口防尘。
