`通信汪CKRS-2型光纤热剥钳`，是用于剥除单芯或者带状光纤涂覆层的工具，能够通过减少操作人员在剥除涂覆层的时间来提高工作效率。

支持多种光纤剥除，包括裸纤、带状光纤、尾纤等；内置6000mAh锂电池，支持Type-C接口充电，加宽的刀片和加热槽，实现光纤秒剥。

## 产品特点

:::tip 
1.	全金属机身，坚实耐用
2.	内置充电锂电池，Type-C充电接口（可用手机等11任何Type-C充电线）
3.	恒温控制系统，开机预热一次，保持恒温，持续秒剥
4.	加宽限位带状刀片，可同时秒剥多种多根光纤，提高工作效率
5.	免调节设计，人体力学结构，剥纤更省力

:::


## 结构说明

![结构图](/assets/image/ckcp/ckcp_1.jpg)

| No.  | 零件名称   |                 功能                 |
| :--- | :--------- | :----------------------------------: |
| A    | 尖端       |       将清洁线引至光连接器端面       |
| B    | 锁定按钮   |         按下按钮可伸缩延长杆         |
| C    | 外壳       |         推动外壳以清洁连接器         |
| D    | 导向盖     | 用作防尘盖或用于清洁光纤跳线上的街头 |
| E    | 余量显示窗 |            剩余清洁线指示            |

  ## 操作步骤

1.长按电源按键，加热显示红灯开机，预热1-2分钟左右，达到恒温状态。

![结构图](/assets/image/ckcp/ckcp_2.jpg)

2.将光纤整齐的排在加热槽中。

![结构图](/assets/image/ckcp/ckcp_3.jpg)

3.盖上上盖，使上下刀片闭紧合实。

![结构图](/assets/image/ckcp/ckcp_4.jpg)

4.平拉光纤（拉光纤方向和刀片方向垂直）。

5.剥纤完成。

**插入适配器时，请注意不要使清洁器倾斜。**

在插入过程中不要过度用力，因为这可能回损坏连接器和清洁笔。

如果推动外壳被阻，请从适配器中取出清洁器，并确保没有大量碎屑阻碍清洁过程。

## 指示灯说明

1.短按电源按键，电量指示灯为绿灯（代表电量充足）、红灯（代表电量不足，即将关机）

![结构图](/assets/image/ckcp/ckcp_5.jpg)

2.长按电源按键，加热灯为红灯代表开机

![结构图](/assets/image/ckcp/ckcp_6.jpg)

3.长按电源按键，加热灯为红灯代表开机。

![结构图](/assets/image/ckcp/ckcp_7.jpg)

**插入适配器时，请注意不要使清洁器倾斜。**

## 充电说明

光纤清洁笔的侧面带有一个透明窗口，用户可以看到到超细纤维卷。这将让您知道在您的剩余用量。
