# CKFC-1 光纤切割刀使用说明

CKFC-1 是用于硅光纤、石英光纤端面切割的精密工具，适用于光纤熔接前的端面制备。本页面根据产品说明书整理，包含结构、参数、标准操作、刀片调节、故障排查及维护方法。

::: warning 安全警告
- 光纤和光纤碎屑非常纤细且尖锐，操作时应佩戴防护眼镜；
- 光纤碎屑必须放入专用容器，不得随意丢弃；
- 不要直接用手触碰刀刃；
- 不要自行拆解切割刀或向内部加油；
- 切割后的光纤端面不得触碰任何物体。
:::

主要部件包括：

| 部件 | 作用 |

| 大压板 | 压紧光纤并配合完成切割 |

| 滑块 | 带动刀片完成横向切割 |

## 切割前准备

准备以下用品：

- CKFC-1 光纤切割刀；
- 与光纤规格匹配的专用夹具；
- 光纤剥纤钳；
- 纯度大于 99% 的酒精；
- 无尘纸、脱脂棉或清洁棉；
- 光纤碎屑收集盒；
- 防护眼镜。

切割前应检查：

1. 刀片、橡胶压垫和光纤导向槽无灰尘、油污；
2. 滑块可以顺畅移动；
3. 刀片锁紧螺钉和高度锁紧螺钉无松动；
4. 光纤夹具与被切光纤规格匹配；
5. 大压板和夹具盖板能够正常开合。

## 标准操作方法

### 1. 打开切割刀

打开大压板和夹具盖板，将安装刀片的滑块推至靠近操作者的一侧。

### 2. 剥除并清洁光纤

1. 使用剥纤钳剥除光纤涂覆层；
2. 预留裸纤长度约 `20-35 mm`；
3. 使用蘸有纯度大于 99% 酒精的无尘纸或脱脂棉包住光纤；
4. 沿同一方向将裸纤擦拭干净；
5. 一张无尘纸或一块脱脂棉只使用一次，不要反复擦拭。

### 3. 放置光纤

左手将光纤放入专用夹具的导向槽内，使裸纤保持笔直，并正确放置在橡胶压垫上。

::: caution
裸纤若弯曲、倾斜或未正确放置在压垫上，会导致端面角度过大、端面突起或缺损。
:::

### 4. 完成切割

1. 合上夹具盖板；
2. 合上大压板；
3. 平稳推动刀片滑块；
4. 让滑块自由移动到另一侧；
5. 刀片划过光纤下表面并完成切断。

不要猛推滑块，也不要在切割中途往回拉动滑块。

### 5. 取出光纤

1. 左手扶稳切割刀；
2. 右手打开大压板；
3. 用左手捏住光纤；
4. 右手打开夹具盖板；
5. 小心取出已经切好的光纤。

切好的端面不得碰触夹具、手指、桌面或其他物体。

## 切割端面判断

![CKFC-1 光纤切割端面示意](/assets/image/ckfc1/ckfc1-cut-quality.png)

| 端面现象 | 判断 |
|---|---|
| 端面平整、无明显倾斜 | 切割良好 |
| 端面角度较大 | 切割不良 |
| 端面有明显突起 | 切割不良 |
| 端面出现缺损 | 切割不良 |

如条件允许，可将光纤放入熔接机，通过熔接机放大画面检查端面质量。

## 日常清洁

应经常使用蘸有酒精的棉棒清洁：

- 上、下压垫的橡胶表面；
- 刀片刃口附近；
- 光纤导向槽；
- 光纤夹具；
- 滑块轨道周围。

当切割效果变差时，应首先进行清洁，再判断是否需要调整刀片。

::: danger
清洁刀片时不要用手直接触碰刃口。只能使用棉棒轻轻擦拭。
:::

## 刀刃位置调节

刀片为圆形刀片，分为 1-16 个使用位置。某个刃口磨损后，可以旋转到下一处新刃口继续使用。

![CKFC-1 刀片锁紧螺钉](/assets/image/ckfc1/ckfc1-blade-lock.png)

操作步骤：

1. 使用扳手松开刀片锁紧螺钉，无需完全取下；
2. 使用棉棒抵住刀片；
3. 缓慢旋转刀片至下一个新位置；
4. 重新拧紧刀片锁紧螺钉；
5. 必须确认螺钉已经锁紧；
6. 试切 1-2 根光纤；
7. 使用熔接机观察端面质量。

::: warning
不要用手直接转动刀片，也不要用镊子或其他金属工具拨动刃口，以免割伤或损坏刀刃。
:::

## 刀片高度调节

![CKFC-1 刀片高度调节螺钉](/assets/image/ckfc1/ckfc1-height-adjustment.png)

1. 将滑块推到切割完成的位置；
2. 松开高度锁紧螺钉；
3. 轻微旋转高度调节螺钉：
   - 顺时针：调高刀片；
   - 逆时针：调低刀片；
4. 每次只调整很小的角度；
5. 调整完成后，拧紧高度锁紧螺钉；
6. 试切光纤并检查端面；
7. 必要时重复微调。

不要一次大幅度调整刀片高度，否则可能损伤光纤、压垫或刀片。

## 常见故障与处理

### 光纤切不断，端面凹凸或有明显斜角

可能原因：

- 光纤未笔直放置在橡胶压垫上；
- 刀片位置磨损；
- 刀片高度过低；
- 刀片、压垫或导向槽有灰尘；
- 滑块移动不顺畅。

处理方法：

1. 重新放置光纤；
2. 清洁刀片、压垫和导向槽；
3. 将刀片旋转到新的刃口位置；
4. 适当调高刀片；
5. 重新试切并检查端面。

### 光纤出现裂纹、端面阴影或角度过大

可能是刀片高度过高。

处理方法：

1. 清洁光纤和夹具；
2. 检查光纤是否笔直；
3. 适当调低刀片高度；
4. 重新试切。

### 刀片调高或调低后仍有较大斜角

可能是光纤端面偶然产生裂纹，且裂纹方向没有对准观察方向。

建议重新剥纤、清洁和切割，再通过熔接机画面从不同方向检查。

## 刀片循环使用

- 圆形刀片共标注 1-16 个位置；
- 理论上圆周每个有效位置可完成约 3000 次切割；
- 1-16 区域可依次使用；
- 16 个位置全部使用后，可在允许范围内适当提高刀片高度，再循环使用；
- 实际寿命会受到光纤材质、清洁状态、操作方法和刀片调节情况影响。

## 刀片更换

1. 使用扳手松开刀片锁紧螺钉；
2. 取下锁紧螺钉和垫片；
3. 打开大压板；
4. 使用镊子夹住刀片两侧；
5. 将旧刀片平稳抬起并移除；
6. 使用镊子夹持新刀片的两侧；
7. 保持刀片水平，将刀片孔套入刀片轴；
8. 使用棉棒抵住刀片，将刀片旋转到 `1` 号位置；
9. 安装垫片；
10. 拧上并锁紧刀片锁紧螺钉；
11. 试切光纤并检查端面；
12. 必要时微调刀片高度。

::: caution
使用镊子时，不得让镊子碰触刀刃，否则可能损伤刃口并影响切割性能。
:::

## 维护与存放

- 使用后及时清理光纤碎屑；
- 保持刀片、夹具和压垫清洁；
- 不要向切割刀内部添加润滑油；
- 不要自行拆解精密部件；
- 避免跌落、撞击、挤压及强烈振动；
- 存放时关闭大压板，避免灰尘进入；
- 放置在干燥、清洁的工具箱内；
- 需要维修时请联系厂家或售后服务人员。
